# Patient Appointment and Emergency Resource Tracker

## Project Overview
This Python project helps clinics or hospitals manage patient appointments and track emergency resources such as medicines and organs. The system provides easy scheduling, resource updates, and viewing functionality to improve healthcare management efficiency.

## Features
- Schedule and view patient appointments with dates.
- Track the quantity of emergency medicines and organs in inventory.
- Update resource stock levels dynamically.
- Simple console-based interface for easy interaction.

## Technologies Used
- Python 3.x for core programming
- Built-in data structures (dictionaries) for storing appointments and resources

## Getting Started

### Prerequisites
- Python 3.x installed on your machine
- Basic familiarity with running Python scripts in a terminal or command prompt

### Usage
- The script allows you to add appointments by providing patient names and dates.
- You can view all scheduled appointments.
- Update the quantity of listed emergency medicines or organs.
- View current stock of all emergency resources.
